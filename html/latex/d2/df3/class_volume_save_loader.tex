\doxysubsection{詳解}
各カテゴリ(Master,Music,SFX.\+UI)の音量を外部ファイルに保存しつつスライダーで調整するクラス スライダー側での値の変更の適用は\+Audio\+Play\+Volume\+Test.\+csで行われている。 各スライダーは設定しなくても動作する(セーブロードのみが可能。)
